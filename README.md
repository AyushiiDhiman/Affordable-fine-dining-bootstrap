# 🍽️ Affordable Fine Dining Restaurant Website

A modern, responsive restaurant website built using Bootstrap 5 featuring Indian and Italian cuisine within an affordable price range (₹400–₹600). The project focuses on using Bootstrap utility classes and components instead of heavy custom CSS while delivering a premium glassmorphism UI experience.

---

## ✨ Features

- 💎 Glassmorphism luxury UI design
- 🎞️ Bootstrap carousel hero section
- 📑 Tab-based menu categories (Indian & Italian)
- 🍛 10+ food items with images
- ⭐ Rating stars for each dish
- 📱 Fully responsive design
- 📌 Sticky animated navbar with shrink effect
- 📂 Offcanvas mobile navigation menu
- 🎥 Scroll animations using AOS
- 📩 Reservation form with modal confirmation popup
- 📞 Contact form section
- 🎨 Fancy typography using Google Fonts

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- Bootstrap Icons  
- AOS (Animate On Scroll)  
- Google Fonts

## 🎯 Project Objective

This project demonstrates practical implementation of Bootstrap 5 components including navbar, carousel, cards, tabs, modal, offcanvas, and forms while maintaining responsive design principles and a clean UI layout.
